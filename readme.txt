== Installation ==

Installing "Hubtel Payment Gateway for WooCommerce" can be done using the following steps:

1. Go to the plugins page of WordPress by clicking 'Add New', search for "Hubtel Payment Gateway for WooCommerce", install and then activate.
2. Alternative Method - Download the plugin from WordPress.org, upload the `wc-hubtel-payment-gateway` folder to your `/wp-content/plugins/` directory via FTP or upload the wc-hubtel-payment-gateway.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer and then activate the plugin.
